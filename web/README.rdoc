== README

BrisAccess

Use the API:

```
/data/locations
```
List of all the accessibility data by location
Can filter by ?suburb=Brisbane

```
/data/qr
```
List of all the accessibility data for Queensland Rail
Can filter by ?station=Albion

```
/data/areas
```
List of all the accessibility data by areas (areas of the Brisbane)
Can filter by ?suburb=Brisbane

```
/data/ferries
```
List of all the accessibility data by ferry stations
Can filter by ?stop=Regatta
